# Roadmap

PosePair / 对比一拍 is currently an early-stage WeChat Mini Program focused on privacy-first before/after comparison photography.

The current Mini Program version is Chinese-first because it is being tested mainly in the WeChat ecosystem. Future iOS and Android versions are planned to support English UI and international distribution.

## Short-term

- Improve README and developer setup guide
- Polish WeChat Mini Program UI
- Improve local image stitching quality
- Add more angle guide templates
- Improve Before photo outline extraction
- Improve package structure for open-source reuse
- Add clearer privacy and local-processing explanations

## Mid-term

- Add local face-outline detection where supported
- Improve export resolution and image quality
- Add more use-case templates for:
  - Medical aesthetics
  - Nail care
  - Beauty salons
  - Hair salons
  - Skincare
  - Fitness documentation
  - Other before/after documentation workflows
- Add better error handling and permission guidance
- Prepare English documentation for international developers
- Collect feedback from real beauty-service practitioners

## Long-term

- Build a free iOS version
- Build a free Android version
- Add English UI support for iOS and Android apps
- Release the iOS version on international App Stores, including the U.S. App Store
- Explore Android distribution for international users
- Add optional local AI-assisted outline tools
- Support more localized templates for beauty-service workflows in different countries and regions
- Keep the project open-source so developers worldwide can adapt the ideas and algorithms to their local industries and user needs
